# Mztracker

## Overview
Mztracker is a task tracking application designed specifically for the MZcet organization. This Flutter project serves as a starting point for developing a robust and user-friendly application to manage tasks efficiently.

## Features
- **Task Creation:** Add new tasks with relevant details.
- **Task Management:** Edit, update, and delete tasks as needed.
- **Task Tracking:** Track the status of tasks to ensure timely completion.
- **User Interface:** Intuitive and clean UI for easy navigation and task management.

## Installation

### Prerequisites

- Flutter SDK
- Android Studio or VS Code with Flutter and Dart plugins
- A connected device or emulator

### Steps

1. **Clone the repository:**
   ```bash
   git clone https://github.com/saravanan2k03/Mztracker.git
   ```

2. **Navigate to the project directory:**
   ```bash
   cd Mztracker
   ```

3. **Get the dependencies:**
   ```bash
   flutter pub get
   ```

4. **Run the application:**
   ```bash
   flutter run
   ```
